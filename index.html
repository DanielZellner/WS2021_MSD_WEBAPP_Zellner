Short summary of the task:

Sie richten sich Ihre Entwicklungsumgebung ein. Dazu verwenden wir in diesem Semester im ersten Schritt einen Webserver und eine Entwicklungsumgebung zum Entwickeln von HTML Seiten. Sie werden im Laufe der Übungen auch einen Datenbankserver benötigen. Daher empfiehlt sich die Verwendung von zB XAMPP.Sie werden auf Ihrer Landing Page ein Thema präsentieren. Sie wollen sich im ersten Schritt noch nicht über das Design Gedanken machen. Sie konzentrieren sich auf den Inhalt und eine valide HTML Struktur. Welche Vorteile besitzt ein valides HTML Dokument? Welche Nachteile können sich bei einem nicht validen HTML Dokument ergeben? Ihre Landing Page gehört strukturiert. Welche Strukturierungsmöglichkeiten sind dazu geeignet? Wie können Sie die Landing Page übersichtlich gestalten? Weshalb ist es wichtig Hierarchien zu beachten? Erstellen Sie für Ihre Landing Page ein einfaches Kontaktformular. Was wird dazu alles benötigt? Wie unterscheiden sich einzelne Eingabefelder? Wieso sollte nicht nur ein einfaches Textfeld verwendet werden? Wozu werden Formulare benötigt? Laden Sie Ihre HTML Seite auf den Übungsserver der FH JOANNEUM alternativ gerne auch auf einen anderen / eigenen Webspace, wenn vorhanden.
